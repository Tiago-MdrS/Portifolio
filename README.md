# Portfólio Dev - Tiago Madeira

Um portfólio moderno e responsivo desenvolvido com React, Vite e Tailwind CSS para apresentar minhas habilidades, projetos e experiências como Desenvolvedor Full Stack.

##  Sobre o Projeto

Este projeto foi criado com o objetivo de centralizar informações sobre minha trajetória como desenvolvedor, destacando tecnologias, projetos desenvolvidos e formas de contato.

O portfólio possui uma interface moderna, limpa e responsiva, proporcionando uma ótima experiência em dispositivos desktop e mobile.

---

##  Funcionalidades

* Página inicial com apresentação profissional
* Seção "Sobre Mim"
* Exibição de habilidades técnicas
* Portfólio de projetos
* Área de contato
* Navegação suave entre seções
* Layout responsivo
* Design moderno utilizando Tailwind CSS
